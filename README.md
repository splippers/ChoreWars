# ChoreWars

ChoreWars is the **meta layer** for chore gamification: profiles, progression, streaks, history, and cross-mode scoring.

## Related repo
- **BoreDoom (Quest 3 AR “doing the chore” module)**: `https://github.com/splippers/BoreDoom`

## Intended split of responsibilities
- **BoreDoom**: Quest 3 passthrough AR modes (vacuuming, mowing, attic/basement/garage/shed), spatial capture, coverage, on-device UX.
- **ChoreWars**: account/profile model, streak rules, scoring progression, achievements, “season” challenges, and any back-end / sync layer (if you choose to add one).
